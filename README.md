## **Installation**

#### Dependencies

run in shell:
```elm
pip install -r requirements.txt
```
__Important:__ depending on your system, make sure to use `python3` instead.


#### Running

```elm
git clone https://github.com/DoktaPola/PhoneBook.git
cd PhoneBook
(run in IDE terminal ->) main.py      
```
